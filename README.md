#### 技术方案：
- 服务端：go-micro
- 数据库：mysql
- 缓    存：redis
- 前   端：el & vue
- 服务器： centos 7 & nginx
- 本地环境：go1.9
- 容器：Docker
- 进程管理：supervisor
